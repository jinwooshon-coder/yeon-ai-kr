Q3 오답 해설

오답 so~that / 관계대명사

문제: 다음 빈칸에 알맞은 것을 고르시오.

The그 problem문제 was~이었다 ___ difficult어려운 ___ nobody아무도 ~않다 could~할 수 있었다 solve풀다, 해결하다 it.그것
나의 답: ② which ... that
→ 관계대명사 which와 접속사 that을 혼동하여 선택
정답: ④ so ... that
→ 「so + 형용사 + that + 주어 + 동사」 구문

핵심 문법: so ~ that 구문 (결과 부사절)

「so + 형용사/부사 + that + S + V」는 "너무 ~해서 …하다"라는 결과를 나타낸다.

The그 movie영화 was~이었다 so boring지루한 that I나는 fell빠지다(fall 과거) asleep.잠든
(영화가 너무 지루해서 잠들었다.)
She그녀는 ran달렸다(run 과거) so fast빠르게 that no아무 ~도 아닌 one사람 could~할 수 있었다 catch잡다 her.그녀를
(그녀가 너무 빨리 달려서 아무도 못 잡았다.)

비교: such ~ that 구문

명사(구)를 수식할 때는 「such + (a/an) + 형용사 + 명사 + that」을 쓴다.

It그것 was~이었다 such a difficult어려운 problem문제 that nobody아무도 ~않다 could~할 수 있었다 solve풀다 it.그것

오답 원인: 관계대명사와의 혼동

관계대명사 which는 앞에 있는 명사(선행사)를 수식하는 형용사절을 이끈다. 이 문장에서는 결과를 표현하는 부사절이 필요하므로 which가 아닌 so ~ that이 정답이다.

구문역할구조
so~that결과 (부사절)so + 형용사 + that + S + V
관계대명사 which수식 (형용사절)명사 + which + V ...

Q19 오답 해설

오답 가정법 영작

문제: 다음 우리말을 영작하시오.

"만약 내가 새라면 너에게 날아갈 텐데."

나의 답: If만약 I나는 am a bird,새 I나는 will fly날다 to~에게 you.너
→ 직설법(현재 시제)으로 작성 — 가정법 과거 미적용
정답: If만약 I나는 were~이라면 (가정법) a bird,새 I나는 would~할 텐데 fly날다 to~에게 you.너
→ 가정법 과거: 현재 사실의 반대를 가정

핵심 문법: 가정법 과거 (Subjunctive Past)

현재 사실과 반대되는 상황을 가정할 때 사용한다.

공식: If + S + 동사의 과거형, S + would/could/might + 동사원형

If만약 I나는 were you,너 I나는 would accept수락하다 the offer.제안
(내가 너라면 그 제안을 수락할 텐데.)
If만약 she그녀가 had more더 많은 time,시간 she그녀는 could finish끝내다 the project.프로젝트
(그녀에게 시간이 더 있다면 프로젝트를 끝낼 수 있을 텐데.)

주의: be동사는 인칭에 관계없이 were

가정법 과거에서 be동사는 주어에 관계없이 were를 쓴다. (구어체에서 was를 쓰기도 하지만, 시험에서는 were가 정답)

직설법 (사실)가정법 과거 (반대 가정)
I am not a bird.If I were a bird, ...
He doesn't have money.If he had money, ...
She can't fly.If she could fly, ...

오답 원인 분석

"만약 ~라면"을 보고 조건문을 만들었지만, 현재 불가능한 상황(사람은 새가 될 수 없음)이므로 가정법 과거를 써야 한다. am → were, will → would로 바꿔야 한다.

구분나의 답정답
if절 동사am ✗were ✓
주절 조동사will ✗would ✓
